Netover Straighterは、「手軽さ」「拡張性」「理論最速」を追求した
次世代の遠隔操作アプリです。

1. インストールもアカウントも不要
アプリのインストールも、面倒なアカウント登録もいりません。

(1) ZIPをダウンロード  
(2) ブラウザを開く  
(3) ペアリングする  

それだけで準備完了。
あとは、いつでもどこでも操作できます。

覚えることはほとんどありません。
必要なのは、鍵を保存した場所くらいです。

2. 通信はいつも「ストレート」に
Netover Straighterは、WebRTCによるP2P通信を採用。

サーバーを経由せず、端末同士が直接接続します。
だからこそ実現できる、理論上最速のレスポンス。

「遅いリモート操作」は、もう過去のものです。

3. さらに拡張できる
このプロジェクトはオープンソースです。

機能を追加するのも、改造するのも自由。
あなたの用途に合わせて進化させることができます。

料金はかかりません。

+--------------------------------------------------------------------+

さて、選ぶのはあなたです。

「重いパソコンを持ち歩く」のか、
それとも「どこからでも接続する」のか。

## Versioning
このプロジェクトはSemVerに従いたいです。
とはいえ、αバージョン、βバージョンの扱いがv1.0.0001以下では変化があります。
- v1.0.0001以下
  - αバージョン、βバージョンの区別はなく、すべてβバージョンとして扱います。
- v1.0.0001以上
  - αバージョンは試験段階、βバージョンは正式リリース前のバージョンとします。
